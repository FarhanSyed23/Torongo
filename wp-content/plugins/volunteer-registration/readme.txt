== Description ==
This is simple volunteer registration plugin.
The User will appear in Users menu. After login user can edit their profiles.
USE echo do_shortcode( 'volunteer_registration_form' ); in template and theme.
USE [volunteer_registration_form] in pages and posts.

== Installation ==
1. Go to the Add New plugins screen in your WordPress Dashboard
2. Click the upload tab
3. Browse for the plugin file (volunteer-registration.zip) on your computer
4. Click "Install Now" and then hit the activate button
